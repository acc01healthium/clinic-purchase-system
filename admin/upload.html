上傳商品 CSV 檔

支援欄位（不分大小寫）：

  • sku（建議唯一，用來判斷是新增或更新）
  • name（商品名稱）
  • category（分類）
  • unit（單位，例如：盒、箱、包）
  • spec（規格，例如：50入 / 100ml）
  • description（說明，可選）
  • price 或 last_price（售價 = 最後採購價）
  • is_active（1 / 0、true / false、是 / 否）

匯入時會：

  • 已有相同 sku → 更新資料（含售價＆價格更新時間）
  • 沒有的 sku → 新增商品
