CSV 批次匯入商品

匯入說明（請使用 UTF-8 CSV）

請將 Excel 表格另存為 UTF-8 編碼 .csv，欄位名稱請依下列格式：

欄位名稱 說明 是否必填
name商品名稱必填
spec規格必填
category分類必填
unit單位（盒、包、瓶…）必填
last_price進價（數字）必填
suggested_price建議售價（數字）選填
is_active啟用狀態（1/0 或 true/false）必填
description描述（長文字）選填

※ 系統會以 name + spec 判斷是否同一商品：
  ▪ 若已存在 → 更新該筆商品
  ▪ 若不存在 → 建立新的商品
